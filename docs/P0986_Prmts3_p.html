Cadastro de Clientes - Aba Parâmetros (3)

  • Dados de investidor estrangeiro:
    • Alíquota IR Investidor Estrangeiro - utilizada no cálculo de IR quando o cliente é um Investidor Estrangeiro.

    • Cliente não residente - Informa se o cliente reside no exterior.
    • Paraíso Fiscal - Informa se o cliente reside em uma localidade considerada paraíso fiscal.
    • País de Residência - Informa o país de residência do Cliente
  • Plano de Fundação - Indica qual plano o cliente pertence.
  • Tributação Especial:
    • Usa Alíquota IR do Sistema - Informa se o cliente irá utilizar as alíquotas de IR do sistema definidas no Navegador de Alíquotas de IR (1395), ao invés de utilizar as alíquotas de IR definidas para o cliente.

    • Permite cadastrar as Alíquotas de IR para os Investidores (Local e Não Residente, PF ou PJ) de forma segregada para Amortização/Resgate e Distribuição de Resultados.
      Para efetuar o cadastro, deverá ser informada a data de vigência das alíquotas e a Categoria de Tributação de IR. Estas alíquotas só poderão ser cadastradas quando o parâmetro Cliente não residente estiver marcado e o parâmetro Usa Alíquota IR do Sistema estiver desmarcado.
  • Dados de Fundo Imobiliário:
    • Cotista Incorporador, Construtor ou Sócio de Empreendimento Imobiliário - Informa se o cotista é um incorporador, construtor ou sócio de empreendimento imobiliário, para que seja possível verificar a concentração de cotas do Fundo em cotistas com tal característica.

      Para determinar a concentração de cotas são considerados, além do próprio cotista, os cotistas ligados a ele. Sendo assim, após parametrizar o cotista como Incorporador, Construtor ou Sócio de Empreendimento Imobiliário, se faz necessário realizar a associação deste com os cotistas a ele ligados através do Navegador de Associação de Clientes.

    • Fundos Associados - Caso o cotista seja parametrizado como incorporador, construtor ou sócio de empreendimento imobiliário, se faz necessário identificar os fundos nos quais o mesmo possui tal característica. Esta associação deverá ser realizada através desta opção.

    • Uma vez realizada esta associação, mesmo que o parâmetro "Cotista Incorporador, Construtor ou Sócio de Empreendimento Imobiliário" seja desmarcado, o sistema não irá desfazer a associação. Contudo o botão "Fundos Associados..." será desabilitado. Ao calcular a carteira, somente serão considerados na verificação de concentração de cotas, os cotistas com o parâmetro "Cotista Incorporador, Construtor ou Sócio de Empreendimento Imobiliário" marcado, e somente nesta situação será necessário identificar os Fundos Associados.
    Para a tributação dos Cotistas em Carteiras classificadas com a Natureza "Fundo Estruturado", devem ser informados os valores para as Alíquotas de IR das Amortizações, Resgates e Distribuições de Resultados, que eventualmente poderão ser processadas por estas Carteiras.

    O cadastro das Alíquotas em questão, deverá ser realizado neste processo ou via seleção da opção Usa Alíquota IR do Sistema/Carteira.

    Porém caso a opção Isento de IR, cadastrada na Aba Parâmetros (2), estiver selecionada as Alíquotas cadastradas não serão consideradas.
