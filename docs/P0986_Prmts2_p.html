Cadastro de Clientes - Aba Parâmetros (2)

  • Dados de isenção de impostos:
    • Isento de IR até 31/12/1994
    • Isento de IOF
    • Isento de IR RV até 31/12/1997
    • Isento de IOF sobre Aplicação
    • Isento de IR RF até 31/12/1997
    • Isento de IR
    • Isento por Liminar ou Optante MP2222
    • Isento de Antecipação de IR
    • Situação - Nenhum, Ativo ou Inativo
    • Código CETIP - permite a definição do código CETIP do cliente
    • Código ISIN - permite a definição do código ISIN do cliente. Apenas disponível quando a opção Fundo ou Clube de Investimento estiver selecionada.
  • Modo de Liquidação Padrão:
    • Aplicação - permite a definição de um modo de liquidação padrão para o movimento de aplicação
    • Resgate - permite a definição de um modo de liquidação padrão para o movimento de resgate
  • Outros dados:
    • Dependente no Informe de Rendimentos/DIRF - define se o cliente é considerado dependente para efeito de emissão do relatório Informe de Rendimentos ou na geração da DIRF
    • Fundo de Renda Fixa Capital Estrangeiro
    • Resgate Bloqueado - quando marcado não permite que o cliente faça resgate e a posição do cliente deixará de apresentar em alguns processos/relatórios
    • Bloqueia Correspondência - bloqueia a geração de correspondência para o cliente
    • Fundo ou Clube de Investimento
    • Aplicação Bloqueada - bloqueia novas aplicações ou entrada de cotas
    • Flag de Usuário
    • Compensa Prejuízo - determina que resgates efetuados pelo cliente poderão compensar prejuízo. Quando desmarcado, o cliente não disponibilizará o seu saldo de prejuízo para que seja compensado por um outro cliente com o mesmo CPF/CNPJ
    • Agrega Prejuízo por CPF/CNPJ - determina que os resgates efetuados pelo cliente poderão compensar prejuízo de outros clientes que possuirem o mesmo CPF/CNPJ e que também tenham este parâmetro marcado
    • Cliente Bovespa - quando marcado, indica que as informações cadastrais deste cliente foram oriundas da Bolsa.
    • Exige Iniciativa da Operação (RDIP) - define a obrigatoriedade de informar a Iniciativa da Operação (RDIP) nas movimentações de cotistas.

      Para obter maiores informações, acesse o link Iniciativa da Operação (RDIP).
