Formatação de Dados

Regras de Formatação Brasileira:

  • Números Decimais: Utilizam vírgula como separador decimal (ex: 1234,56)
  • Datas: Formato dd/MM/yyyy (ex: 15/03/1980)
  • Data/Hora: Formato dd/MM/yyyy HH:mm:ss (ex: 10/01/2025 14:30:25)
  • CPF: Formato 999.999.999-99 (ex: 123.456.789-00)
  • CNPJ: Formato 99.999.999/9999-99 (ex: 12.345.678/0001-90)
  • CEP: Formato 99999-999 (ex: 12345-678)
  • Valores Boolean: "SIM" ou "NÃO"
  • Valores Nulos: Campos numéricos vazios são exportados como "0"
  • Campos de Texto: Campos vazios são exportados como string vazia ""
  • Separador: Ponto e vírgula (;) entre todos os campos
  • Escape: Campos contendo separadores ou aspas são delimitados por aspas duplas

Observações Importantes

  • O arquivo é gerado com encoding ISO-8859-1 para compatibilidade com sistemas legados
  • A primeira linha contém sempre o cabeçalho com os nomes dos campos
  • Todos os 63 campos são sempre exportados, mesmo quando vazios
