Cadastro de Clientes - Aba Parâmetros

  • Dados do Co-Titular:
    • Nome
    • Pessoa Jurídica - se não estiver marcado o Tipo de Pessoa será Física
    • CPF/CNPJ (conforme o tipo de pessoa)
  • Classe de Investidor

  • Datas de Aniversários:
    • Titular
    • Co-Titular
    Se o cliente for Pessoa Jurídica:

    • Tipo de Pessoa Jurídica - apresenta os tipos disponíveis
    • Imunidade - quando marcado significa que o cliente é uma entidade imune
    • Entidade Imune - apresenta os tipos de entidades disponíveis
    • Código Galgo - Código do Cliente no Sistema Galgo
  • Perfil CVM
    • Usar Perfil CVM Atualizado - utiliza novo conjunto de perfis CVM para identificação do tipo de Investidor (B3)
    • Perfil CVM - indica a classificação do cliente para efeito de emissão do relatório Perfil Mensal CVM.

  • Dados de Tipo de Custódia:
    • Tipo de Custódia - Nenhuma, Interna ou Externa
    • Carteira Associada - Código da carteira a qual o cliente está associado. Esta informação será utilizada para a criação automática de títulos de RC12. Seu preenchimento será obrigatório caso o tipo de custódia seja Interna. Uma vez definida, não poderá mais ser alterada pelo usuário
    • Captura Dados da Carteira Associada - disponível quando uma carteira associada é definida, este campo quando marcado faz com que parte dos dados da aba Endereço e as contas correntes da aba Informações Bancárias sejam preenchidos automaticamente com os dados correspondentes da carteira associada. Nenhuma dessas informações poderá ser editada. Quando alteradas na carteira associada, o Sistema automaticamente atualiza o cliente
    • Resp. Boletagem - indica o player responsável pela boletagem
    • Cliente Custódia - indica o player cliente de custódia
    • Custodiante - indica o player custodiante
    • Distribuidor - indica o player distribuidor
    • Segmento - Indica o segmento do cotista a ser utilizado na Exportação de Movimentação Consolidada.
  • Categoria do Investidor (CVM 554/2014)
    • Profissional - Conforme Art. 9º A da Instrução CVM 554
    • Qualificado - Conforme Art. 9º B da Instrução CVM 554
