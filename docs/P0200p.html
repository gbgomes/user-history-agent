Descrição:

O Cadastro de Processo de Importação de Cadastro de Clientes lhe permite cadastrar os processos que serão utilizados na Importação de Cadastro de Clientes. Para incluir um Processo de Importação de Cadastro de Clientes no cadastro é necessário digitar o código e o nome do processo e especificar o layout do arquivo a ser usado. Isto implica em definir, para cada dado, sua posição(coluna) e seu tamanho(n.º de caracteres).

Sendo que os dados de Cliente, Nome, Assessor são obrigatórios e os outros são opcionais. Obs.: O campo “Cliente = Agência + Conta”, ao ser selecionado, tornará obrigatório o preenchimento dos campos Agência e Conta.

O campo referente à Posição do parâmetro Status é de preenchimento opcional. Se não estiver configurado neste processo, os clientes são importados com o status Ativo no Cadastro de Clientes.

Caso opte pelo preenchimento do campo “Posição” correspondente ao parâmetro Status, o usuário deve definir os literais correspondentes aos três tipos de status (Ativo, Inativo e Encerrado). Se um dos literais for preenchido, é obrigatório o preenchimento dos demais. Caso contrário, é apresentada uma mensagem de validação. Além disso, os referidos campos devem ser diferente um do outro. Caso não sejam, também é apresentada uma mensagem de validação.

Campo DV Conta só será utilizado se a flag Grava Informação Bancária estiver marcada no processo 203 - Importação do Cadastro de Clientes.

*Atenção: Informação apenas cadastral, aguardando liberação de suporte no processamento das Amortizações, Resgates e Distribuição de Resultados.
Conforme a legislação vigente, a tributação ocorrerá de forma linear considerando a alíquota vigente na data do processamento (Sem Corte).
