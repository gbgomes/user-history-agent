Cadastro de Clientes

Parâmetros:

Exportação de Implantação

Descrição:

A Exportação de Cadastro de Clientes tem por objetivo gerar um arquivo com dados de clientes.
Caso seja marcado a opção Exportação de Implantação será exportado o cadastro completo de clientes e a tag "tipo-geracao-arquivo" será preenchida com a palavra "implantacao".
Caso contrário o tipo de geração gravado nesta tag será "manual".

Formato do arquivo exportado - Versão 1.00.
Formato do arquivo exportado - Versão 2.00.
